### 一、简介

> PlayerScoreboard —— 一款有点好用的玩家计分板插件

1. 支持不同权限不同计分板
2. 支持多计分板配置
3. 计分板支持变量
4. 计分板支持指定世界开/关
5. 完善的计分板相关 API
6. 服务器版本1.13及以上无行长限制（使用1.12及以下版本时，行数限制为32个字符，包括颜色编码）
7. 在服务器1.16及以上版本上，支持完整的RGB格式
8. 在服务器版本1.20.3或更高版本中，可以在侧边栏隐藏红色数字
9. 支持MiniMessage(1.1.0+)
