### 1. Introduction

> PlayerScoreboard - A somewhat useful player scoreboard plugin

1. Support for different scoreboards based on permissions
2. Support for multiple scoreboard configurations
3. Scoreboard supports variables
4. Scoreboard supports world-specific on/off
5. Comprehensive scoreboard-related API
6. No line length limit on server version 1.13 and above (when using version 1.12 and below, line limit is 32
   characters, including color codes)
7. Full RGB format support on server version 1.16 and above
8. Hide red numbers on sidebar in server version 1.20.3 or higher
9. Support MiniMessage (1.1.0+)
